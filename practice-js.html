What is JavaScript?

JavaScript, often called JS, is a programming language that adds interactivity, logic, and dynamic behavior to a website. While HTML creates the structure of a webpage and CSS defines the style and appearance, JavaScript is what makes the page active and responsive to user actions. For example, HTML can display a button, CSS can make it look red and round, and JavaScript can make it show an alert when clicked, validate a form before submission, or change content dynamically without reloading the page.

JavaScript is not limited to web browsers. With technologies like Node.js, it can also run on servers, but for now, we focus on its use in websites. Unlike HTML and CSS, which are declarative (they tell the browser what something is), JavaScript is imperative and procedural, meaning it tells the computer step by step what to do and how to respond to events. This is why modern websites rely heavily on JS to handle interactive features such as sliders, pop-ups, dropdowns, animations, and live updates.
